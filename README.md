# Miropad

This is the public homepage for Miropad — a view-only web surface for AI-generated content resources.

## Overview

Miropad provides read-only access to AI-generated content resources including professional reports, audits, strategy documents, and analysis sheets. Content is accessible through secure, view-only links without editing capability.

## Deployment

This static site is deployed to Vercel and serves as the public-facing homepage for the Miropad service.

## Technology

- Pure HTML/CSS (no build process required)
- Minimal, infrastructure-focused design
- Mobile-responsive layout
- System font stack for native performance

## Local Development

Simply open `index.html` in a web browser. No build step or local server required.
